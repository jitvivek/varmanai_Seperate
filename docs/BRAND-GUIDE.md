# VarmanAI Brand Guide

## Brand Identity

**VarmanAI** (वर्मन् = shield/armor in Sanskrit) — AI security for India and the world.

## Colors

| Token | Hex | Usage |
|-------|-----|-------|
| Deep Indigo | `#0A1330` | Background, primary dark |
| Electric Cyan | `#22D3EE` | Accents, CTAs, active states |
| Gold | `#F5B544` | Highlights, premium indicators |
| Slate 200 | `#E2E8F0` | Body text |
| Slate 400 | `#94A3B8` | Secondary text |
| Red 500 | `#EF4444` | Danger, threats |
| Green 500 | `#22C55E` | Success, safe |

## Typography

| Role | Font | Weight |
|------|------|--------|
| Display / Headings | Sora | 600, 700 |
| Body | Plus Jakarta Sans | 400, 500, 600 |
| Code / Monospace | JetBrains Mono | 400, 500 |

## Logo Usage

- Primary: Full wordmark on dark backgrounds
- Mark: Shield icon only for favicons and small contexts
- Minimum size: 24px height for mark, 80px width for wordmark
- Clear space: 1x height of the shield mark on all sides

## Voice & Tone

- **Confident** but not arrogant
- **Technical** but accessible
- **India-first** — reference Indian languages, local compliance (DPDPA, RBI)
- **Action-oriented** — verbs over nouns, concise sentences

## Extension UI

- Popup width: 360px
- Border radius: 8-16px
- Shadow DOM for content overlays (no style leakage)
- Animations: 150-200ms ease transitions
